Wicket Flickr Demo

This application demonstrates a use of a new Wicket feature: partial page rendering for Ajax type applications. It uses the Ruby on Rails video demo application which caused quite a stirr in the Web Application community.

Although this is a fully functional demo, it depends on currently experimental features of Wicket and the techniques used may not be considered a best practise.

Before you start

Please provide the application with your own Flickr API Key, otherwise the service won't accept your requests. You can set this key in the web.xml deployment descriptor, using the 'flickr_api' init parameter to the Wicket servlet.

The application will not contact Flickr, but instead use pre-defined search results. The following search keys are available:

  • flower
  • color
