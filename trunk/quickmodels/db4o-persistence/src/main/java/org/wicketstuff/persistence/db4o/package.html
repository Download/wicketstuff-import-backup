This library implements the Persistence Facade library's database interface over the db4o object database.

The db4o implementation defines an API of two additional interfaces which applications may use to access db4o-specific functionality:

  • Backup - back up the database to another file on disk
  • Defrag - accesses db4o's defragmentation facility
To create and start using a database, create an instance of the Db class over db4o as follows: 
            Db database = Db4oDb.create ("path/to/database/file");
