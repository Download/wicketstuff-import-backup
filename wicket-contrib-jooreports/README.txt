Wicket Contrib JooReports 
=========================

  Extending OOPage class you can generate a Wicket page that output 
 a report from an OpenOffice document template and the page data
 Model. 
 
  It uses jooreports library to generate the document and convert 
 it with OpenOffice (working as a service) into one of the possible 
 openoffice export formats. (pdf, word, excel...).
